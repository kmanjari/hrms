Editor example Editor advanced initialisation examples

The examples in this section show more advanced initialisation and configuration techniques than the simple examples. Examples here show more complex join operations, using REST interfaces and how to show data in the table or form only, among others.

These examples, along with the API examples will help you get the most out of Editor.

Advanced initialisation

  • Data shown only in the form
  • Data shown in table only
  • Complex (nested) JSON data source
  • REST interface
  • Ajax override - using localStorage for the data source
  • Row ID source specification
  • DOM sourced table
  • Export buttons
  • Join tables - self referencing join
  • Join tables - link table
  • Join tables - one-to-many join

Please refer to the DataTables documentation for full information about its API properties and methods.
Additionally, there are a wide range of extras and plug-ins which extend the capabilities of DataTables.
