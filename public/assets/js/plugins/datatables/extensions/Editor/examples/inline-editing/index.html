Editor has three different Editor modes:

  • Primary editing
  • Bubble editing
  • Inline editing

The examples in this section show the inline editing option, which displays the field to be edited inside the table cell to be edited. This is fantastic for rapid editing of data in a table, very much like a spreadsheet application.
